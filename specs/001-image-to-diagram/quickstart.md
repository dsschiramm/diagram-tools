# Quickstart: Image to Diagram

1. Open a terminal in `apps/image-to-diagram`.
2. Run `npm install`.
3. Run `npm run dev`.
4. Configure the provider, endpoint, model, and optional API key.
5. Paste a public JPG, PNG, BMP, or SVG URL.
6. Select a diagram format and click **Add**.
7. Click **Process**.
8. Review the Markdown preview.
9. Use **Save**, **Open**, or **Copy** as needed.

Provider settings are saved to `~/.image-diagram/config.json`.
